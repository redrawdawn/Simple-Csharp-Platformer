# Simple C# Platformer

A small Godot 4 C# pixel-art platformer prototype with multiple levels, coins, hats, power-ups, enemies, moving platforms, and a boss fight.

## How to Open

1. Install Godot 4.6 Mono/.NET.
2. Open `project.godot`.
3. Press Play.

## Controls

- A / Left Arrow: move left
- D / Right Arrow: move right
- Space: jump
- F: shoot after collecting the fireball power-up

## Notes

The exported Windows build is not stored in this repository. Re-export from Godot when needed.
